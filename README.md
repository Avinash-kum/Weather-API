# SkyPulse - Weather API

A real-time weather application built with React and Vite. Get instant weather information, hourly forecasts, and 5-day weather predictions for any location worldwide.

## Features

- 🌍 Search weather by city name
- 📍 Get weather using your current location
- 🌡️ Toggle between Celsius and Fahrenheit
- 🔑 Easy API key management
- ⏰ Hourly weather forecast
- 📅 5-day weather forecast
- 📊 Detailed weather information

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
4. Run the development server: `npm run dev`
5. Enter your API key in the app

## Technologies

- React 19
- Vite
- OpenWeatherMap API
- CSS3
